Meccanica
Механика
Физические величины, законы Ньютона (включая всемирное тяготение), энергия, количество движения и механика жидкостей.
Скалярные и векторные величины
Grandezze scalari e vettoriali
Скалярные: масса, время, температура, энергия Векторные: сила, скорость, ускорение, перемещение

Скалярные — только число и единица.
Векторные — модуль, направление и смысл.

Кинематика
Cinematica
v = Δx / Δt
a = Δv / Δt
x = x₀ + v₀t + ½at²
Законы Ньютона
Leggi di Newton
F = m · a (II закон)
F₁₂ = −F₂₁ (III закон)
F_g = G · m₁m₂ / r² (тяготение)
G = 6.674 × 10⁻¹¹ Н·м²/кг²

I закон (инерция): тело покоится или движется равномерно, если F_рез = 0.

Импульс и мощность
Quantità di moto e potenza
p = m · v (импульс)
F = Δp / Δt
P = W / t = F · v (мощность)

Закон сохранения импульса: в замкнутой системе Σp = const.

Энергия и работа
Energia e lavoro
W = F · d · cos θ
Ek = ½mv²
Ep = mgh
Механика жидкостей
Meccanica dei fluidi
P = F / S (давление)
P = P₀ + ρgh (гидростатика)
F_Arch = ρ_жидк · g · V (Архимед)
P₁ + ½ρv₁² + ρgh₁ = P₂ + ½ρv₂² + ρgh₂

Последнее — уравнение Бернулли. Тело плавает если ρ_тела ≤ ρ_жидкости.

Ottica
Оптика
Геометрическая оптика: отражение, преломление, зеркала, линзы, призмы. Системы линз.
Законы отражения и преломления
Riflessione e rifrazione
θᵢ = θᵣ
n₁ sin θ₁ = n₂ sin θ₂ (Снеллиус)
n = c / v

Полное внутреннее отражение: sin θc = n₂/n₁ (при n₁ > n₂). При θ > θc — луч не проходит.

Сферические зеркала
Specchi concavi e convessi
1/f = 1/dₒ + 1/dᵢ
f = R / 2
M = −dᵢ / dₒ

Вогнутое (concavo): f > 0, собирающее. Выпуклое (convesso): f < 0, рассеивающее. dᵢ < 0 → мнимое изображение.

Тонкие линзы
Lenti sottili (concave e convesse)
1/f = 1/dₒ + 1/dᵢ
D = 1/f (диоптрии, f в метрах)

Собирающая (convessa): f > 0. Рассеивающая (concava): f < 0.
Системы линз: 1/f_tot = 1/f₁ + 1/f₂.

Призмы и электромагнитный спектр
Prismi e spettro EM

Призма: свет преломляется на двух гранях; n зависит от длины волны → дисперсия (разложение в спектр).

Спектр EM: радио → микроволны → ИК → видимый → УФ → рентген → γ-лучи.
c = λ · ν, E = hν (h = 6.63×10⁻³⁴ Дж·с).

Termodinamica
Термодинамика
Температура, теплота, теплоёмкость, расширение тел, газовые законы и начала термодинамики.
Температура и теплота
Temperatura e calore
Q = m · c · ΔT
T(K) = T(°C) + 273.15

c — удельная теплоёмкость (Дж/(кг·К)). Вода: c = 4186 Дж/(кг·К). Теплота передаётся от горячего к холодному (II начало).

Тепловое расширение
Dilatazione termica
ΔL = α · L₀ · ΔT (линейное)
ΔV = β · V₀ · ΔT (объёмное)

α — коэффициент линейного расширения; β ≈ 3α для твёрдых тел. Газы расширяются значительно больше твёрдых и жидких тел.

Идеальный газ
Gas ideale
pV = nRT
R = 8.314 Дж/(моль·К)
p/T = const (при V=const)
pV = const (при T=const, Бойль)
V/T = const (при p=const, Шарль)
I и II начала термодинамики
Principi della termodinamica
ΔU = Q − W

I начало: энергия сохраняется. Q — теплота, поглощённая системой; W — работа, совершённая системой.

II начало: теплота самопроизвольно переходит от горячего к холодному. Энтропия замкнутой системы не убывает.

η = 1 − T₂/T₁ (КПД Карно)
Elettromagnetismo
Электромагнетизм
Электростатика, конденсаторы, закон Ома, магнитное поле, электромагнитные волны.
Закон Кулона и электрическое поле
Legge di Coulomb — Campo elettrostatico
F = k · q₁q₂ / r²
k = 9 × 10⁹ Н·м²/Кл²
E = F / q = k·Q / r²
V = k·Q / r (потенциал)

Работа поля: W = q·ΔV. E = −ΔV/Δd между пластинами.

Конденсаторы
Condensatori
C = Q / V
C = ε₀ · A / d (плоский)
E_конд = Q²/(2C) = CV²/2

Последовательно: 1/C = 1/C₁ + 1/C₂. Параллельно: C = C₁ + C₂.

Закон Ома и цепи
Legge di Ohm — Circuiti
V = I · R
P = V · I = I²R = V²/R

Последовательно: R = R₁+R₂. Параллельно: 1/R = 1/R₁+1/R₂. Интенсивность тока I = Q/t (Кл/с = А).

Магнитное поле и сила Лоренца
Campo magnetostatico — Forza di Lorentz
F = qvB sin θ
F = BIL sin θ (на проводник)

B — магнитная индукция (Тл). Направление F: правило правой руки. Магнитное поле прямого тока: B = μ₀I/(2πr).

Chimica
Химия & Строение вещества
Строение атома, периодическая таблица, типы связей, ионные и молекулярные соединения, вода и атмосфера.
Строение атома
Struttura atomica
Протоны (Z) — заряд ядра Нейтроны (N) — в ядре Электроны — оболочки
A = Z + N (массовое число)

Изотопы: одинаковый Z, разный N. Электронная конфигурация определяет химические свойства.

Периодическая таблица
Tavola periodica degli elementi
Периоды → заполнение оболочек Группы → одинаковая валентность

Слева направо: металлический характер убывает, электроотрицательность растёт. Сверху вниз: атомный радиус растёт, электроотрицательность убывает.

Главные группы: IA (щелочные металлы), IIA (щёлочноземельные), VIIA (галогены), VIII/0 (благородные газы).

Типы химических связей
Legami chimici
Ионная: металл + неметалл Ковалентная: неметалл + неметалл Металлическая: металл + металл Водородная: H···F, O, N

Ионные соединения: кристаллическая решётка, высокие T плавления, проводят ток в растворе.
Молекулярные: более низкие T плавления.

Вода и атмосфера
Acqua e atmosfera
H₂O: угол H–O–H ≈ 104.5°

Вода — полярная молекула, водородные связи → аномально высокая T кипения (100°C).

Атмосфера: N₂ (~78%), O₂ (~21%), Ar (~1%), CO₂ (~0.04%). CO₂ — парниковый газ. O₃ (озон) в стратосфере поглощает УФ.

Stechiometria
Стехиометрия
Моль, молярная масса, уравнивание реакций, стехиометрические расчёты, символика химических формул.
Моль и молярная масса
Quantità di sostanza
n = m / M
N = n · Nₐ
Nₐ = 6.022 × 10²³ моль⁻¹

M — молярная масса (г/моль), m — масса в граммах. Молярный объём идеального газа при н.у.: Vm = 22.4 л/моль.

Химические символы и формулы
Simbologia chimica

Атомный символ: ᴬ_Z X (A — масса, Z — заряд).
Молекулярная формула: H₂O, CO₂, NaCl.
Уравнение реакции: реагенты → продукты, закон сохранения масс.

Коэффициенты уравниваются так, чтобы число атомов каждого элемента совпадало с обеих сторон.

Стехиометрические расчёты
Calcoli stechiometrici

Схема расчёта:
1. Записать и уравнять реакцию.
2. Найти n реагента: n = m/M.
3. По коэффициентам найти n продукта.
4. Найти m продукта: m = n · M.

Пример: 2H₂ + O₂ → 2H₂O
4 г H₂ = 2 моль → 2 моль H₂O = 36 г.

Концентрация раствора
Concentrazione
C (моль/л) = n / V(л)
ω (масс. доля) = m_в-ва / m_р-ра
C = ω · ρ · 1000 / M

Soluzioni & pH
Растворы & pH
Кислоты и основания (теория Брёнстеда), pH, шкала кислотности, диссоциация, буферные растворы.
Кислоты и основания Брёнстеда
Acidi e basi (Brønsted)

Кислота: донор протона H⁺ (отдаёт H⁺).
Основание: акцептор протона H⁺ (принимает H⁺).

Сильные кислоты (полная диссоциация): HCl, H₂SO₄, HNO₃.
Слабые: CH₃COOH, H₂CO₃.
Сильные основания: NaOH, KOH. Слабые: NH₃.

Шкала pH
Scala del pH
pH = −log[H⁺]
pOH = −log[OH⁻]
pH + pOH = 14 (при 25°C)
Kw = [H⁺][OH⁻] = 10⁻¹⁴

pH < 7: кислота. pH = 7: нейтрально. pH > 7: щелочная среда.

Степень диссоциации и Ka
Grado di dissociazione
α = n(диссоц.) / n₀
Ka = [H⁺][A⁻] / [HA]

Ka — константа кислотности. Чем больше Ka, тем сильнее кислота. pKa = −log Ka. Для слабой кислоты: [H⁺] ≈ √(Ka · C).

Нейтрализация и соли
Neutralizzazione e sali
HCl + NaOH → NaCl + H₂O

Соли образуются при реакции кислоты с основанием. Соль сильной кислоты и сильного основания → pH = 7. Соль слабой кислоты → pH > 7 (гидролиз).

Algebra
Алгебра
Линейные и квадратные уравнения, теорема Виета, прогрессии, свойства корней.
Линейное уравнение
Equazione di primo grado
ax + b = 0 → x = −b/a

Если a = 0, b ≠ 0: нет решений. Если a = 0, b = 0: любое x — решение (уравнение тождественно).

Квадратное уравнение и теорема Виета
Equazione di secondo grado — Viète
x = (−b ± √(b²−4ac)) / 2a
D = b² − 4ac
x₁ + x₂ = −b/a
x₁ · x₂ = c/a

D > 0: два различных корня. D = 0: один корень. D < 0: нет вещественных корней.

Арифметическая прогрессия
Progressione aritmetica
aₙ = a₁ + (n−1)d
S = n(a₁ + aₙ) / 2
Геометрическая прогрессия
Progressione geometrica
aₙ = a₁ · qⁿ⁻¹
S = a₁(1 − qⁿ) / (1 − q)
S∞ = a₁ / (1 − q) (|q| < 1)
Analisi matematica e funzioni
Функции и Анализ
Понятие функции, область определения, пределы, производные стандартных функций, интегралы.
Понятие функции и ОДЗ
Concetto di funzione

Функция f: X → Y ставит каждому x ∈ X единственное y ∈ Y.

ОДЗ (dominio): значения x, при которых f(x) определена.
Типичные ограничения: подкоренное выражение ≥ 0; знаменатель ≠ 0; аргумент логарифма > 0.

Производные стандартных функций
Derivate fondamentali
(xⁿ)′ = n·xⁿ⁻¹
(eˣ)′ = eˣ
(ln x)′ = 1/x
(sin x)′ = cos x
(cos x)′ = −sin x
(tan x)′ = 1/cos²x
(aˣ)′ = aˣ · ln a
Правила дифференцирования
Regole di derivazione
(f·g)′ = f′g + fg′
(f/g)′ = (f′g − fg′)/g²
(f∘g)′ = f′(g(x)) · g′(x)

Последнее — правило цепочки (chain rule).

Integrale

∫eˣdx = eˣ + C

∫sin x dx = −cos x + C
∫cos x dx = sin x + C

Геометрия
Планиметрия, стереометрия и аналитическая геометрия: прямая, окружность, конические сечения.

Segmenti, angoli, luoghi geometrici

Trigonometria
Тригонометрия
Основные тождества, формулы сложения, двойного и половинного угла, теоремы для треугольников.
Основные тождества
Identità fondamentali
sin²x + cos²x = 1
tan x = sin x / cos x
1 + tan²x = 1/cos²x
1 + cot²x = 1/sin²x
Формулы сложения (addizione)
Formule di addizione
sin(α±β) = sinα cosβ ± cosα sinβ
cos(α±β) = cosα cosβ ∓ sinα sinβ
tan(α±β) = (tanα ± tanβ)/(1 ∓ tanα tanβ)
Формулы двойного угла
Formule di duplicazione
sin 2α = 2 sinα cosα
cos 2α = cos²α − sin²α
cos 2α = 2cos²α − 1 = 1 − 2sin²α
tan 2α = 2tanα / (1 − tan²α)
Формулы половинного угла
Formule di bisezione
sin²(α/2) = (1 − cosα) / 2
cos²(α/2) = (1 + cosα) / 2
tan(α/2) = sinα / (1 + cosα)
Теоремы синусов и косинусов
Teoremi su triangoli
a/sin A = b/sin B = c/sin C = 2R
c² = a² + b² − 2ab cos C

Statistica & Probabilità
Статистика
Описательная статистика, гистограммы, комбинаторика, вероятность и случайные события.
Средние и показатели разброса
Valori medi e dispersione
x̄ = Σxᵢ / n (среднее)
σ² = Σ(xᵢ − x̄)² / n (дисперсия)
σ = √σ² (стандартное откл.)

Мода: наиболее частое значение. Медиана: центральное значение при упорядоченном ряду.

Частоты и гистограммы
Frequenze e istogrammi
f_i = n_i / n (относит. частота)

Гистограмма: по оси X — значения (или интервалы), по оси Y — частота (или относительная частота). Сумма всех частот = 1 (= 100%).

Диаграмма частот: столбик для каждого значения. Площадь пропорциональна частоте. Читать: наиболее высокий столбик = мода.

Вероятность
Probabilità
P(A) = m / n
P(A∪B) = P(A) + P(B) − P(A∩B)
P(A|B) = P(A∩B) / P(B)

Независимые события: P(A∩B) = P(A)·P(B). Несовместные: P(A∩B) = 0.

Комбинаторика
Combinatoria — permutazioni e combinazioni
Pₙ = n! (перестановки)
Cₙᵏ = n! / (k!(n−k)!)
Aₙᵏ = n! / (n−k)! (размещения)

Cₙᵏ = "n choose k" — число способов выбрать k из n без учёта порядка.

Aritmetica e Algebra
Числа & Арифметика
Свойства целых, рациональных и вещественных чисел. Модуль. Степени и корни. Логарифмы.
Множества чисел
Insiemi numerici
ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ

ℕ — натуральные, ℤ — целые, ℚ — рациональные (дроби p/q), ℝ — вещественные (включают иррациональные, например √2, π).

Модуль числа
Valore assoluto
|x| = x если x ≥ 0; −x если x < 0
|x| < a ⟺ −a < x < a
|x| > a ⟺ x < −a или x > a

Свойства: |ab| = |a|·|b|; |a+b| ≤ |a|+|b| (неравенство треугольника).

Степени и корни
Potenze e radici
aᵐ · aⁿ = aᵐ⁺ⁿ
(aᵐ)ⁿ = aᵐⁿ
a⁻ⁿ = 1 / aⁿ
ⁿ√a = a^(1/n)
ⁿ√(a·b) = ⁿ√a · ⁿ√b
Логарифмы & Показательные функции
Logaritmi ed esponenziali
logₐ(x) = y ⟺ aʸ = x
log(a·b) = log a + log b
log(a/b) = log a − log b
log(aⁿ) = n · log a
logₐ(b) = log b / log a

ln x = logₑ x (натуральный), log x = log₁₀ x (десятичный). eˡⁿˣ = x.

Algebra
Многочлены (Polinomi)
Операции с многочленами, буквенные выражения, разложение на множители, схема Горнера.
Буквенные выражения
Calcolo letterale
(a+b)² = a² + 2ab + b²
(a−b)² = a² − 2ab + b²
(a+b)(a−b) = a² − b²
(a+b)³ = a³ + 3a²b + 3ab² + b³
Разложение на множители
Decomposizione in fattori
a³ − b³ = (a−b)(a²+ab+b²)
a³ + b³ = (a+b)(a²−ab+b²)

Методы: вынесение за скобку, группировка, формулы сокращённого умножения, теорема Безу (если p(r)=0, то (x−r) — делитель).

Деление многочленов (схема Горнера)
Divisione tra polinomi / Ruffini

Для деления p(x) на (x−r): записываем коэффициенты, умножаем и складываем по схеме. Остаток = p(r).

Пример: x³−6x²+11x−6 ÷ (x−1):
1 | −6 | 11 | −6
→ 1 | −5 | 6 | 0 (делится нацело)

Operazioni con polinomi

Умножение: каждый член на каждый.
Алгебраические дроби: сокращать только на общие множители числителя и знаменателя.

Квадратные неравенства
Disequazioni di secondo grado
ax² + bx + c > 0

1. Найти корни x₁, x₂ (x₁ < x₂).
2. Если a > 0: решение x < x₁ или x > x₂.
3. Если a > 0 и D < 0: неравенство выполняется всегда (∀x).
4. Паrabola volta su: positiva esternamente alle radici.

Дробно-рациональные уравнения
Equazioni razionali fratte
P(x)/Q(x) = 0

Условие: P(x) = 0 И Q(x) ≠ 0.

ОДЗ: Q(x) ≠ 0 — обязательно проверять.
Умножать обе части на Q(x) нельзя без проверки знака (важно для неравенств).

Уравнения и неравенства с радикалами
Equazioni e disequazioni con radicali
√f(x) = g(x)

Условия: f(x) ≥ 0 И g(x) ≥ 0.
Тогда возводим в квадрат: f(x) = g(x)².

Для √f(x) < g(x): нужно g(x) > 0 И f(x) < g(x)².

√(√x) = x^(1/4)
Показательные и логарифмические уравнения
Equazioni logaritmiche ed esponenziali
aˣ = aʸ ⟺ x = y
log_a(f) = log_a(g) ⟺ f = g, f > 0

Для логарифмических неравенств: знак меняется если 0 < a < 1 (основание). Всегда проверять ОДЗ (аргумент > 0).

Тригонометрические уравнения
Equazioni trigonometriche
sin x = a → x = arcsin a + 2πk
x = π − arcsin a + 2πk
cos x = a → x = ±arccos a + 2πk
tan x = a → x = arctan a + πk

k ∈ ℤ. Особые случаи: sin x = 0 → x = πk; cos x = 0 → x = π/2 + πk.

Параллельные: m₁ = m₂. Перпендикулярные: m₁ · m₂ = −1.

Окружность
Circonferenza
(x−a)² + (y−b)² = r²
x² + y² + Dx + Ey + F = 0

Центр (a,b), радиус r. В общей форме: центр (−D/2, −E/2), r = √(D²/4 + E²/4 − F).

Парабола
Parabola
y = ax² + bx + c
Вершина: x_v = −b / (2a)
y_v = c − b²/(4a)

a > 0: ветви вверх; a < 0: вниз. Ось симметрии x = −b/(2a).

Эллипс и Гипербола
Ellisse e Iperbole
Эллипс: x²/a² + y²/b² = 1
Гипербола: x²/a² − y²/b² = 1

Эллипс: a,b — полуоси. Если a=b — окружность. Гипербола: асимптоты y = ±(b/a)x.

Уравнивание ОВР (метод e⁻)
Bilanciamento redox

1. Определить степени окисления всех атомов.
2. Выявить, кто окисляется (+), кто восстанавливается (−).
3. Записать полуреакции окисления и восстановления.
4. Уравнять число электронов (умножить на коэффициенты).
5. Сложить полуреакции и подобрать коэффициенты.
